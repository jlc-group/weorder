# Global Agent Rules - Copy to Antigravity User Settings

เปิด **Open Antigravity User Settings** (⇧⌘ล) แล้ว copy ข้อความด้านล่างไปใส่ใน `userRules` section:

---

```json
{
  "userRules": [
    "ก่อนทำอะไร ต้องบอกก่อนว่าจะทำอะไร - ไม่ทำเงียบๆ",
    "ถ้าคำสั่งถูก cancel หรือ error ต้องบอกทันที - ไม่เงียบ",
    "ไม่ทำอะไรเองโดยไม่ถามก่อน - ยกเว้นสิ่งที่สั่งชัดเจน",
    "ถ้าทำไม่ได้ ต้องบอกว่าทำไม่ได้ - ไม่รอนานแล้วเงียบ",
    "ห้าม git revert/reset/checkout โดยไม่ถาม",
    "ห้ามลบไฟล์โดยไม่ถาม",
    "ห้ามแก้ไข .env หรือ config สำคัญโดยไม่ถาม",
    "ห้ามสมมติ - ต้องเช็คข้อมูลจริงก่อนพูด",
    "ห้ามทำหลายอย่างพร้อมกัน - ทำทีละขั้น verify ก่อนไปต่อ",
    "ห้ามพูด 'ขอโทษ' โดยไม่แก้ปัญหา",
    "ห้ามบอกให้ผู้ใช้ทำแทน",
    "ถ้าทำผิด ต้องกู้คืนทันทีและบันทึกลง .agent/error_log.md"
  ]
}
```

---

## ขั้นตอน:

1. กด **⇧⌘ล** หรือเปิด **Open Antigravity User Settings**
2. หา section `userRules` 
3. เพิ่มกฎเหล่านี้
4. Save

กฎจะ apply กับทุก workspace โดยอัตโนมัติ
